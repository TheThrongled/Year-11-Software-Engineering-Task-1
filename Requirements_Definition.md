# Requirements Definition

The main aim of the software is to create an *interactive educational product for software engineering students*
And also **include an intuitive web browser interface that displays directions and responds to user actions**

FUNCTIONAL REQUIREMENTS
- Users can create accounts, log in and log out.
- Users can recover their accounts through email connection
- Users can access interactive tutorials
- Users are given feedback to their inputs and work.
- Users can view progress
- The user can use menus to navigate the software
- The web interface responds to layout size and device model.
